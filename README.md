## ✨ Основные возможности

* **⚔️ Динамическая боевая система HTML5 Canvas**: Высокопроизводительный 60 FPS рендеринг сражений с анимациями ударов, заклинаний, критических уронов, метеорных дождей, молний и системой частичек.
* **🛡️ Кукла Снаряжения (Paperdoll RPG)**: Удобная анатомическая панель экипировки героического класса (Оружие, Шлем, Броня, Наплечники, Плащ, Штаны, Знамя, Перчатки, Наколенники, Сапоги, Амулет, Кольца, Серьги).
* **💎 Процедурный лут-генератор (1M+ предметов)**: Уникальная система аффиксов и суффиксов, 7 рангов редкости (Обычный, Необычный, Редкий, Эпический, Легендарный, Мифический, Божественный) с нативными сравнениями мощи и бессмертными историями-легендами для каждого предмета.
* **🐉 Эпические арт-токены Боссов и Монстров**: Высокодетализированные иллюстрации Боссов и Мини-боссов с руническими боевыми пьедесталами, коронами `👑` и уникальным поведением.
* **🌟 Глубокое древо талантов и скиллов**: 24 уникальных таланта с ветками развития и 12 боевых заклинаний с автокастом.
* **🗺️ 16 Игровых зон и Тайные подземелья**: Продвижение по этапам, циклы мастерства локаций и походы в опасные данжи за легендарными сокровищами.
* **📜 Динамические квесты и поручения**: Награды за убийство монстров, прохождение этапов и боссов с автоматическим отслеживанием прогресса.
* **💾 Автосохранение и адаптивный интерфейс**: Мгновенное сохранение в `localStorage`, поддержка умного позиционирования окон (Smart Tooltips) и полная адаптация под любые разрешения.

## 🎨 Дизайн и интерфейс

Интерфейс приложения спроектирован по канонам современного **Dark Fantasy RPG UX**:
* **Стекломорфизм и Neumorphism**: Полупрозрачные подложки, глубокие тени `backdrop-blur-md` и мягкое свечение артефактов.
* **Цветовая палитра редкостей**: Ослепительное свечение редких предметов (зелёный, синий, фиолетовый, золотой, огненный и божественный платиновый).
* **Smart Window Tooltips**: Умная система всплывающих карточек сравнения снаряжения, предотвращающая любой вылет элементов за пределы видимого экрана.

## 🎮 Игровой процесс и механики

### 🛡️ Экипировка и Инвентарь
При клике по любому слоту на кукле персонажа, инвентарь автоматически отфильтрует все имеющиеся предметы под выбранный тип снаряжения. Подсказки сравнивают урон, броню и здоровье с текущим надетым предметом, подсвечивая прирост зелёными стрелочками `▲`.

### 📜 Легенды предметов
Каждый найденный предмет обладает уникальным сгенерированным описанием-историей — от баллад о городских кузнецах до космогонических саг о древних богах.

## 🛠️ Технологический стек

* **Core**: React 18 / TypeScript / Vite 5
* **Graphics & FX**: HTML5 2D Canvas API + Custom Particle Engine
* **Styling**: TailwindCSS 3 + Custom Glassmorphism Utilities
* **State & Persistence**: Zustand Store + LocalStorage Synchronizer

## 💻 Сборка и локальный запуск

### 1. Клонирование репозитория:
```bash
git clone https://github.com/ReiKatari/STORM_CHRONICLES.git
cd STORM_CHRONICLES
```

### 2. Установка зависимостей:
```bash
npm install
```

### 3. Запуск сервера разработки:
```bash
npm run dev
```

Откройте браузер по адресу [http://localhost:3000](http://localhost:3000) (или указанному в консоли порту).

### 4. Сборка production-версии:
```bash
npm run build
```
